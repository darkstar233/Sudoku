# 数独

## 为什么写这个程序？

在北京上小学时，每周三最后一节是课外活动课，可以选择自己喜欢的活动。

我当时选择的是趣味数学，还是我们班主任的课，开始学习的是魔方，一周学一个步骤。

等魔方学完了，又开始教我们数独。

其实也算不上教，就是让我们自己玩，每次给一张数独，让自己慢慢解。

在一个周三，和朋友放学回家的路上，我就在想能不能直接查找到答案，因为只要按照规则判断就可以得到答案。

于是我们就在手机上查，果然有。我们把要求的数独填上去，一下就得到答案了，当时就觉得很厉害。

在我19年高考完，填报志愿的时候，我江龙哥给我参考的。他那一年毕业，学的计算机，当时计算机正是热门专业，于是也给我报了计算机相关志愿，但因分数不够，被第五志愿——智能电网信息工程录取了。当时选这个专业就是因为有“信息工程”四个字。

上学了才发现属于电气工程学院，但是电力方面没有电气工程及其自动化专业精，但是学的杂，因为加进去了部分计算机内容，但是因为课时原因也不精。

我还记得上学报到的时候，我们专业27人，一个班，电气5个班，我当时就给我爸打电话，问专业是不是报错了，人这么少，是不是被骗了？当时电气和我们还在自动化院，上了半年，成立电气工程学院（住上新宿舍、用上新电气大楼），就俩专业，从水利水电学院迁过来不少人并到电气，电气变成了9个还是10个班了。

上大学时，学习了C语言，我立刻被吸引了，不知道是因为我之前很少接触电脑（没去过网吧），还是因为一切符合逻辑，亲自设计就像解密一样。C语言考试99分，满绩。

学C语言的时候我就想起了小时候数独的事情了，想自己亲自设计一个数独求解程序。

但是太懒了，一直没有付出实践。

23年进入国网工作，25年8月因个人原因被放到供电所，值班时闲来无事，买了一台笔记本电脑，又有点电子阳痿，不想打游戏，遂着手设计数独求解程序。

其实在我写这段话的时候（26年01月14日23点整），数独求解的程序早就完成了。

只不过完成之后觉得有点不满足，想要再把各种数独的求解以及玩法全部写进来。

我向来奉行大就是好的理念。

现在是设计界面阶段，完善逻辑，把各部分功能拆分到各个文件里，作为模块功能。

其他数独的求解和生成需要再等一段时间。

至于为什么现在才写这段话，是因为这两天才学会用git，才想着要稍微正规一点。

完。（26年01月14日23点07分）
